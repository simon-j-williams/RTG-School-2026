# RTG-School-2026

This repository contains a set of Jupyter notebook exercises for a short introduction to quantum computing. The material covers core concepts in quantum computing, as well as selected topics including quantum machine learning and Hamiltonian simulation.

The notebooks are designed to be run interactively using Qiskit, and include a mix of guided examples and exercises.

---

## Getting started

It is recommended to use a virtual environment to avoid conflicts with your existing Python setup.

### 1. Create a virtual environment

```bash
python -m venv qc_env

source qc_env/bin/activate

pip install -r requirements.txt
